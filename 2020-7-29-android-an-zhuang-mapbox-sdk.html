运行

警告：Compatible side by side NDK version was not found. Default is 21.0.6113669.

虽然运行成功了，但是出现

Compatible side by side NDK version was not found. Default is 21.0.6113669.

警告。

解决方案：

IDEA工具栏：

Tools->SDK Manager

Android SDK页面，SDK Tool，选择NDK(Side by Side)

再勾选右下角 Show package details

勾选对应版本

点击 OK 开始下载

完成后重新运行项目
