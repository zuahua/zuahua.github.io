3、默认参数

  • 函数在定义时，就给参数一个默认值
  • 注意：函数定义时，给某个参数默认值，其之后不能出现没有默认值的参数

4、序列参数、字典参数（即将序列，字典传入函数）

  • 列表参数：所有参数作为一个列表传入
    • 在调用时列表前使用一个*号
    • 列表的参数个数与函数参数个数一致
  • 字典参数：所有参数作为一个字典传入
    • 在调用时列表前使用两个**号
    • 字典的参数个数与函数参数个数一致
    • 字典的键与函数形参相同，键作为字符串类型
    • 字典的值传给键所对应的形参
Value of a is one
Value of b is two
Value of c is three
Value of a is one
Value of b is two
Value of c is three

5、星号元组参数

  • 星号元组参数
  • 语法：
    • def 函数名(*元组形参):
      语句块
  • 含义： *元组形参相当于将参数设置成列表
    • 形参可以任意个数
进入元组参数
()
进入元组参数
('1', '2', '3')

6、双星号字典参数

  • 语法：
    • def 函数名(**元组参数):
      函数体
  • **函数名 代表的是字典
进入字典
{}
  • 双星号字典参数，调用时必须按照字典的格式；有”键”和”值”
进入字典
{'a': 'one', 'b': 'two', 'c': 'three'}

7、单星号元组参数与双星号字典参数都叫做 不定长参数

  • 说明：
    • 位置形参，单星号元组形参，双星号字典形参可以混用
    • 混用时，顺序必须是：位置形参，单星号元组形参，双星号字典形参

8、函数返回值

  • 一般需要返回值，使用return
  • 函数的类型就是返回值的类型
  • 可以没有return，不建议去掉
skr skr skr!!!
  • 有return无返回值时，返回None
skr skr skr!!!
None
  • 没有return时，返回值也为None
第二个 skr,没有return
None
  • 返回值为多个的用法
    • 当函数有多个返回值却给它一个存储变量时，自动将其打包为tuple类型
    • 可以将多个返回值赋值给对应个数的变量
    • 所以多个返回值只能打包成一个tuple或者与其对应数量的变量
